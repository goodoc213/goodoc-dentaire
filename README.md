# goodoc-dentaire

Dental medicine lesson repository for the GooDoc app.

## Structure

```
goodoc-dentaire/
├── year1/
│   └── 2024-2025/        <- promo folder (you create this)
│       └── ana-path/     <- subject folder (you create this)
│           └── file.pdf
├── year2/
├── year3/
├── year4/
└── year5/
```

## How to add content

1. Go to the correct yearX/ folder
2. Create a promo folder: 2024-2025/
3. Inside it, create one folder per subject: pharmaco/, ana-path/, etc.
4. Drop your PDF files inside each subject folder

The app reads the structure dynamically - no code changes needed.

## Naming rules
- Lowercase only
- Use - instead of spaces (e.g. ana-path, not Ana Path)
- PDF files only
